# ShortyBatou's Portfolio

Welcome to my personal portfolio website! This is a simple, clean, and responsive website built with HTML and CSS.

## Features

- 📱 Fully responsive design - works on desktop, tablet, and mobile
- 🎨 Modern, professional styling with smooth animations
- 🧭 Easy navigation with sticky header
- 📂 Sections for Home, About, Projects, and Contact
- 🚀 Fast and lightweight

## Structure

```
.
├── index.html      # Main website page
├── styles.css      # Styling and layout
└── README.md       # This file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the content with your information

## Customization

To personalize your website:

1. **Update the HTML content** in `index.html`:
   - Change "Welcome to My Portfolio" to your own title
   - Update the About section with your bio
   - Add your actual projects with descriptions
   - Update contact links

2. **Customize colors** in `styles.css`:
   - Modify the CSS variables at the top of the file (`:root` section)
   - `--primary-color`: Main accent color
   - `--secondary-color`: Secondary accent color
   - `--text-color`: Text color

3. **Update contact links**:
   - Replace the placeholder links in the Contact section with your actual profiles
   - Update the email address
